# Sentiment-analysis-using-RNN
Comparing one hot encoding and word embedding s for IMDb sentiment analysis using RNN
Built and compared two deep learning models for IMDb movie review sentiment classification using TensorFlow.

• Implemented One-Hot Encoding + Dense Network achieving ~50% accuracy
• Developed Embedding Layer + SimpleRNN achieving 77% accuracy  
• Demonstrated 27% performance gain using word embeddings over sparse representation
• Analyzed impact of semantic vector learning and sequence modeling in NLP

Tech Stack: Python, TensorFlow, Keras, NumPy, Matplotlib
